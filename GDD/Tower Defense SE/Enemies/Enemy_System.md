# Enemy System

## System Purpose
The enemy system defines the units that travel through the path and pressure the player's defenses.

## Core Enemy Attributes
- Health
- Speed
- Armor / resistance
- Reward value
- Path behavior
- Special trait
- Vulnerability

## Enemy Design Goals
- Force tower variety
- Create tactical adaptation
- Encourage upgrades
- Increase challenge over time

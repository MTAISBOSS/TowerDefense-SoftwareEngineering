# Level Design

## Level Design Goals
- Support strategic placement
- Create readable path and build-node relationships
- Increase complexity across environments
- Encourage different tower mixes

## Number of Initial Levels
3 core levels

## Level Types
- [[Levels/Level_01_Grassland]]
- [[Levels/Level_02_Canyon]]
- [[Levels/Level_03_Fortress]]

## Map Design Principles
- Clear enemy path visibility
- Distinct tower build spots
- Choke points for area damage towers
- Long lanes for ranged towers
- Open and narrow spaces mixed together

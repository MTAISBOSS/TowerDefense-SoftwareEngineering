# Resources and Economy

## Main Resource
Gold

## Gold Sources
- Starting gold at level start
- Enemy kill rewards
- Bonus reward after wave completion
- Optional flawless wave bonus

## Gold Sinks
- Tower placement
- Tower upgrades
- Optional tower sell penalty system

## Economy Goals
- Encourage planning
- Reward efficient defense
- Prevent tower spam
- Make upgrade timing important

## Sample Economy Values
- Starting Gold: 300
- Basic tower cost: 100
- Advanced tower cost: 150–200
- Upgrade Level 1 cost: 75
- Upgrade Level 2 cost: 125
- Upgrade Level 3 cost: 200

## Economy Design Notes
The proposal explicitly mentions managing resources for building and upgrading towers, so economy should be central but simple.

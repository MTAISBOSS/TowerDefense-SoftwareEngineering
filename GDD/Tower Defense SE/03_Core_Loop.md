# Core Loop

## Primary Gameplay Loop
1. Start level
2. Prepare defense during build phase
3. Spend resources to place towers
4. Start wave
5. Towers attack enemies automatically
6. Enemies move along path toward goal
7. Gain rewards from defeating enemies
8. Upgrade or add towers
9. Repeat for next wave
10. Win by surviving all waves or lose if health reaches zero

## Secondary Loop
- Learn enemy traits
- Adjust tower composition
- Optimize tower positions
- Improve upgrade timing
- Handle stronger enemies and changing vulnerabilities

## Moment-to-Moment Decisions
- Which tower to build
- Where to place it
- Whether to save or spend resources
- Which tower to upgrade first
- How to respond to fast, tanky, or swarm enemies

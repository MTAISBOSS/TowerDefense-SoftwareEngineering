# UI / UX Design

## UI Goals
- Clear game state visibility
- Fast and easy tower placement
- Readable tower and enemy information
- Low friction interaction

## Main HUD Elements
- Player health
- Gold
- Current wave number
- Remaining enemies in wave
- Score
- Pause button
- Wave start button
- Speed control button

## Tower Placement UX
- Hover preview on valid tile
- Red/green placement feedback
- Cost shown before placement
- Tower range preview
- Clear error messages for invalid placement

## Tower Interaction UX
- Click tower to open info panel
- Show damage, range, fire rate, level
- Upgrade button
- Sell button
- Target priority selector

## Feedback Systems
- Hit effects
- Damage numbers optional
- Health bars on enemies
- Upgrade confirmation
- Base hit warning

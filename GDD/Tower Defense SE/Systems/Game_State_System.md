# Game State System

## Purpose
Controls the high-level flow of the game.

## States
- Main Menu
- Loading
- Build Phase
- Combat Phase
- Paused
- Victory
- Defeat

## Responsibilities
- Switch state safely
- Notify UI
- Lock/unlock player actions
- Trigger transitions

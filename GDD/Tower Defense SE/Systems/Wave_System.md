# Wave System

## Purpose
Spawns enemies in controlled sequences and manages combat pacing.

## Responsibilities
- Define wave composition
- Spawn enemy groups
- Track active enemies
- Determine wave completion
- Escalate difficulty over time

## Wave Parameters
- Enemy type
- Count
- Spawn delay
- Entry timing
- Wave reward
- Special modifiers

## Pattern Suggestion
Factory Pattern for enemy creation

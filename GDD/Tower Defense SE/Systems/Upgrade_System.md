# Upgrade System

## Purpose
Improves towers through stat increases or special enhancements.

## Responsibilities
- Show upgrade options
- Apply cost
- Improve stats
- Unlock advanced effects

## Upgrade Design
- 3 upgrade levels per tower
- Optionally linear for prototype
- Branching upgrades can be future scope

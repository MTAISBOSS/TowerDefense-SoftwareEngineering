# Placement System

## Purpose
Handles selecting, previewing, validating, and placing towers.

## Responsibilities
- Detect valid build nodes
- Show placement preview
- Validate resource availability
- Create tower instance
- Prevent invalid placement

## UX Requirements
- Visual valid/invalid feedback
- Range preview
- Cost preview
- Fast selection flow

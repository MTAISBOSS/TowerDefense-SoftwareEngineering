# Project Overview

## Title
Tower Defense Game with Unity

## Genre
Strategy / Tower Defense

## Platform
Unity

## Proposal-Based Summary
The project is a Tower Defense game where the player defends a target by placing and upgrading towers against enemy waves. The game must provide smooth, understandable gameplay and implement the core systems of the genre.

## Requirements Derived from the Proposal
- Core Tower Defense mechanics
- Tower placement on map positions
- Tower upgrades
- Strategic defense against enemy waves
- Enemies with different speed, power, and number
- Resource management for building/upgrading
- Progressive level difficulty
- Win/lose logic
- UI for score, health, and status
- Multiple levels/environments

## Core Promise
A readable, modular, and replayable Tower Defense game emphasizing strategic placement and scalable Unity system design.

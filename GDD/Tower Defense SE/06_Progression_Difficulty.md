# Progression and Difficulty

## Progression Structure
- Levels contain multiple waves
- Each new wave increases pressure
- Each new level introduces new enemy combinations
- Stronger enemies appear over time
- Vulnerabilities and enemy interactions evolve across stages

## Difficulty Drivers
- Higher enemy health
- Faster enemy movement
- Larger enemy groups
- Mixed enemy roles in the same wave
- Boss waves
- Enemies with resistances or vulnerabilities

## Difficulty Curve
### Early Game
- Teaches basic placement and wave handling
- Mostly simple enemies

### Mid Game
- Introduces enemy roles and tower specialization
- Requires upgrades and placement optimization

### Late Game
- Mixed waves
- Stronger resistances
- High resource pressure
- Stronger punishment for poor planning
